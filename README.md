# Floricultura

Projeto desenvolvido para a disciplina de **Interação Humano-Computador (IHC)**. O site simula uma floricultura online, apresentando um catálogo de flores organizado em cards, com foco em usabilidade e experiência do usuário.

## Conceitos Aplicados

- Design centrado no usuário  
- Princípios de usabilidade  
- Organização da informação  
- Interface responsiva com Bootstrap  
- Feedback ao usuário no envio de formulário
